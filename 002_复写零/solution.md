[1089. 复写零](https://leetcode.cn/problems/duplicate-zeros/)

简单

提示

给你一个长度固定的整数数组 `arr` ，请你将该数组中出现的每个零都复写一遍，并将其余的元素向右平移。

注意：请不要在超过该数组长度的位置写入元素。请对输入的数组 **就地** 进行上述修改，不要从函数返回任何东西。

**示例 1：**

```
输入：arr = [1,0,2,3,0,4,5,0]
输出：[1,0,0,2,3,0,0,4]
解释：调用函数后，输入的数组将被修改为：[1,0,0,2,3,0,0,4]
```

**示例 2：**

```
输入：arr = [1,2,3]
输出：[1,2,3]
解释：调用函数后，输入的数组将被修改为：[1,2,3]
```

**提示：**

- `1 <= arr.length <= 104`
- `0 <= arr[i] <= 9`

```c++
class Solution {
public:
    void duplicateZeros(vector<int>& arr) {
        int cur = 0, dest = - 1, sz = arr.size();
        while (dest < sz) {
            if (arr[cur]) {
                ++dest;
            }
            else {
                dest += 2;
            }
            if (dest >= sz - 1) {
                break;
            }
            ++cur;
        }
        if (dest >= sz) {
            arr[sz - 1] = 0;
            dest -= 2;
            --cur;
        }
        while (cur >= 0) {
            if (arr[cur] == 0) {
                arr[dest--] = 0;
                arr[dest--] = 0;
            }
            else {
                arr[dest--] = arr[cur];
            }
            --cur;
        }
    }
};
```

1. **定位复写后的末尾索引**
	用双指针 `cur`（原数组遍历指针）和 `dest`（复写后数组的虚拟末尾索引）模拟复写过程。遍历 `arr`，遇到非零元素则 `dest` 前移 1 位，遇到零则前移 2 位。当 `dest ≥ sz-1` 时停止，并记录最终的 `cur` 和 `dest`。
2. **处理边界溢出**
	若 `dest == sz`（说明最后一个零只复写了一次而超出），手动将数组末尾元素置为 0，并将 `dest` 回退 2 位，`cur` 回退 1 位，避免后续复写出错。
3. **从后向前原地复写**
	从记录的 `cur` 开始向左遍历原数组，若当前元素是 0，则从 `dest` 位置向左连续写入两个 0；若不是 0，则将该元素复制到 `dest` 位置。每次写入后 `dest` 递减，`cur` 递减，直到完成所有复写。
